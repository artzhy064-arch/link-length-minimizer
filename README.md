# link-length-minimizer
Сервис для сокращения ссылок на базе ASP.NET Core 8.0 Minimal APIs. Система генерирует короткие идентификаторы для длинных URL, сохраняет их в базу данных и выполняет автоматическое перенаправление.
Функциональные возможности
Сокращение ссылок: Генерация уникального 6-символьного кода для любого валидного URL.
Перенаправление: HTTP 302 Redirect при обращении к короткому адресу.
Автоматическая очистка: Фоновый процесс удаляет устаревшие записи из базы данных по истечении заданного времени.
Локальное хранение: Использование SQLite для минимизации требований к инфраструктуре.
